# NodeJS MVC API

## Getting Started

### Prerequisites
- Node.js
- MongoDB

### Installation
1. Clone the repository:
    ```bash
    git clone https://github.com/your-repo/nodejs-mvc-api.git
    cd nodejs-mvc-api
    ```

2. Install dependencies:
    ```bash
    npm install
    ```

3. Create a `.env` file:
    ```plaintext
    DB_URI=mongodb://localhost:27017/worko
    ```

4. Run the server:
    ```bash
    npm start
    ```

### Running Tests
